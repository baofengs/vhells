# Vhells

Vhells 是 `Vue Whells` 的缩写，一个 Vue UI 组件库。旨在通过造 Vue UI 组件轮子，提升以下技能：

1. 深入理解 Vue 功能；
2. 掌握多种 Vue 组件开发模式；
3. 做好一个开源项目；
4. 重构、TDD/BDD、设计模式。
